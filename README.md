SURVEY TOOL - HƯỚNG DẪN SỬ DỤNG

==============================

1. GIỚI THIỆU
   ==============================

Đây là công cụ hỗ trợ tự động đánh giá khảo sát.

Tool sẽ:

* Tự mở trình duyệt
* Tự đánh giá theo lựa chọn
* Có thể tự submit

Không cần cài đặt thủ công phức tạp.

==============================
2. YÊU CẦU HỆ THỐNG
===================

* Windows 10 hoặc Windows 11
* Có kết nối Internet lần đầu
* Không cần cài Playwright
* Nếu chưa có Node.js, tool sẽ tự cài

==============================
3. CÁCH SỬ DỤNG
===============

Bước 1:
Giải nén file zip

Bước 2:
Mở thư mục vừa giải nén

Bước 3:
Double click vào:

```
start.bat
```

Bước 4:
Đợi tool khởi động

Tool sẽ tự:

* Kiểm tra Node.js
* Cài thư viện nếu cần
* Mở trình duyệt
* Hiển thị bảng Survey Tool

==============================
4. LẦN CHẠY ĐẦU TIÊN
====================

Ở lần chạy đầu tiên:

Tool có thể:

* Tải thư viện
* Cài đặt Node.js
* Cài đặt trình duyệt

Thời gian có thể mất:

```
1 - 3 phút
```

Các lần sau sẽ chạy nhanh hơn.

==============================
5. NẾU TRÌNH DUYỆT KHÔNG MỞ
===========================

Hãy thử:

1. Đóng tool
2. Chạy lại start.bat
3. Kiểm tra Internet

Hoặc liên hệ hỗ trợ.

==============================
6. CẤU TRÚC THƯ MỤC
===================

Không xóa các file sau:

* node_modules
* chrome-profile
* start.bat
* main.js

Nếu xóa các file này tool có thể không chạy.

==============================
7. LƯU Ý QUAN TRỌNG
===================

* Không đổi tên thư mục
* Không di chuyển file riêng lẻ
* Không xóa chrome-profile
* Không chạy trực tiếp main.js

==============================
8. CẬP NHẬT TOOL
================

Khi có phiên bản mới:

1. Xóa thư mục cũ
2. Giải nén bản mới
3. Chạy lại start.bat
